Day 1: JavaScript Drum Kit

A fun exercise in events and listeners, this definitely helped me brush up on some ES6 I didn't know about in the slightest. Given the emphasis on ES6 adherence in this course I should be re-learning much more as we go.

Day 2: CSS + JS Clock

Never thought I'd figure out the clock hands/clock face offset issue after changing their heights! There seem to be some signs of improvement already. Made a lot more changes here based on personal taste, had some ideas and decided to run with them. Nothing earthshattering though.

Day 3: Playing with CSS Variables and JS

There was a smorgasbord of new information I learned in this one, being that I've never touched CSS variables. They are MAGICAL. I feel embarrassed to say I haven't been using them at all, but they were pretty easy to pick up. Shoutouts to Wes for making the information easily digestable and the learning experience (dare I say it?) fun!

Day 4: Array Cardio Day 1

Get those buns in shape! After building up too much array flab it was inevitable that I had to work it off. I ached in neurons I didn't even know I had!

...my awful workout metaphors aside, this one gave me some ample opportunity for critical thinking in .filter() and .sort(), two methods I had yet to hone. Also learned out how to MAKE TABLES IN THE CONSOLE FROM OBJECT DATA USING console.table(). BRUH.

Day 5: Flex Panels Image Gallery

HEY. I THOUGHT THIS WAS JAVASCRIPT30 NOT CSS30. Real talk, there were quite a few clever bits of info in this exercise regarding flexbox. Something that stuck out the most for me was how WELL CSS and JS work together when separation of concerns is kept at the forefront of one's process. Using pseudo-classes, all I had to do to change how the top/bottom words were animated was toggle a class. Always more to hone!

Day 6: Ajax Type Ahead

Ajax requests that don't use $.getJSON()?! I kid you not, it can be done, and it isn't nearly as messy as one would think. Old jQuery habits die hard, though...the urge was strong to simplify for the sake of less code in my face. But hey, that's no way to progress! Vanilla or bust!
